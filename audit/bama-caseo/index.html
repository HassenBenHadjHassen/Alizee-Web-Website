L'essentiel en 30 secondes

Caseo Bama dispose d'atouts solides : une offre menuiserie complète, 130 avis clients Google, une certification Qualibat RGE et un positionnement « fabrication française » fort. Mais Google ne voit pas ce que vos visiteurs voient. Les mots-clés locaux absents de vos titres, les 0 liens dans vos textes, et les images sans description privent votre site du trafic qu'il mérite. Ce rapport détaille exactement quoi corriger, dans quel ordre, et pourquoi.

⚠️ Ce que vous perdez aujourd'hui
  • 🔴 400–800 visites/mois manquées (requêtes locales non couvertes)
  • 🔴 200–400 recherches "MaPrimeRénov fenêtre" / mois non captées
  • 🔴 Trafic Google Images = 0 (100% des images sans texte alternatif)
  • 🔴 Pages /qualité/ et /sav/ orphelines (0 lien entrant depuis les produits)

Audit détaillé par page

Constat : Votre page d'accueil ne contient aucune mention de Vernon dans son titre principal ni dans la balise Title affichée dans Google. C'est comme un cabinet médical sans adresse sur sa plaque.

<title>La menuiserie 100% française - CASEO BAMA</title>
💡 Impact SEO : Google ne vous associe pas à "menuisier Vernon" (150-300 rech./mois)
✅ Recommandation : Reformuler : "Menuiserie Vernon | Fenêtres, Volets, Cuisines — Caseo Bama"

Constat : Vos 12 catégories produits sont listées mais aucun paragraphe de texte ne contient un seul lien interne. Google distribue le "jus SEO" via ces liens — sans eux, vos pages produits restent faibles.

Aucun <a href> dans les balises <p> du contenu
💡 Impact SEO : Pages produits non renforcées par la page d'accueil
✅ Recommandation : Ajouter 5-6 liens dans les textes : "fenêtres sur mesure", "volets", "certifiés RGE" → URLs respectives

Constat : Vos 130 avis Google sont affichés via un widget JavaScript externe (TrustIndex). Google lit difficilement le contenu dynamique : vos avis n'existent pas pour lui.

<div id='trustindex-widget'> (JS externe non indexé)
💡 Impact SEO : Signal de confiance E-E-A-T non transmis à Google
✅ Recommandation : Dupliquer la note et le nombre d'avis en HTML statique : "4.9/5 · 130 avis Google" avec lien

Constat : Logo, certifications Qualibat, photos de projets — toutes vos images ont l'attribut alt vide. Google Images ne peut pas vous indexer, et les robots ignorent totalement vos visuels.

alt="" sur TOUTES les balises <img> du site
💡 Impact SEO : Trafic Google Images = 0. Signal thématique nul pour les robots
✅ Recommandation : Logo : "Menuiserie Caseo Bama Vernon". Certifs : "Certification Qualibat RGE menuiserie". Projets : décrire matériau + lieu

Constat : Votre site utilise uniquement un schéma "Organization" basique. Il manque le type "LocalBusiness" qui permet à Google d'afficher vos horaires, adresse et étoiles directement dans les résultats de recherche.

"@type":"Organization" uniquement dans le LD+JSON
💡 Impact SEO : Pas de rich results (horaires, note, adresse) dans Google
✅ Recommandation : Remplacer par LocalBusiness avec : adresse, téléphone, horaires, géo, areaServed

Constat : La page d'accueil contient environ 300 mots de texte réel. Pour un hub de menuiserie locale, Google s'attend à 600-800 mots minimum couvrant les catégories de produits.

Paragraphe intro (~80 mots) + H3 catégories sans description
💡 Impact SEO : Contenu "thin" — risque de sous-classement
✅ Recommandation : Ajouter 350-400 mots : 2 phrases sous chaque catégorie + intro enrichie

Constat : Fenêtre, volet, cuisine, véranda… listés avec titre + lien uniquement. Google ne peut pas évaluer la pertinence de ces pages depuis l'accueil faute de texte contextuel.

✅ Recommandation : Ajouter 1-2 phrases sous chaque H3 : "Fenêtres PVC, alu et bois sur mesure — isolantes et personnalisables."
